## v0.6.6（2026-06-14）

### 新增
- **新增仅展示默认配置模式**
  - **部署端环境变量控制**：支持通过 `VITE_SHOW_DEFAULT_CONFIG_ONLY=true` 开启；Docker 部署可使用 `SHOW_DEFAULT_CONFIG_ONLY=true` 在运行时注入。仅在已配置默认 API URL 或默认代理时生效，未提供默认服务商配置时不会限制前端配置切换。
  - **锁定配置与服务商选择**：开启后设置页中的“当前配置”和“服务商类型”下拉菜单不可点击。
  - **受限导入 URL 行为**：通过页面 URL 参数传入的配置只会覆盖当前配置字段，不会新建配置、切换服务商类型或导入自定义服务商，避免绕过默认配置锁定。
